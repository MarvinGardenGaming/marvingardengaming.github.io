Space News Hub

  • Technical Description: spacenewshub.com is a HTML5 website that uses CSS3 and SASS for
    styling. It also uses Bootstrap, Flexbox, and CSS Grid to help visualize items on the
    page. Some of the functionality is written in Javascript, but a good portion of the
    data is being pulled through REST API's and RSS feeds. Lastly this website uses twitters
    API to pull data in directly from twitter. The point of creating this website was to
    give people (including myself) a place to get up to date space news without having to
    to read the opinions of publishers with click bait titles. All information on this site
    comes directly from the source.
  • Platform: Web, Mobile Web
  • Release Date: 17 January 2020
  • Technologies: HTML5, CSS3, JS, JQuery, Bootstrap, Flexbox, CSS Grid, REST, Twitter API
  • Visit Here: www.spacenewshub.com

Astro Repair

  • Technical Description: Astro Repair is a game developed in the latest Unity 2018 LTS version. The graphics were created using the pixel art software called Aseprite. The scripting was done in C#. This was my first Unity game. I had previously done some basic tutorials for Unity but had never actually developed a game. I created this game in a 48 hour game jam called the "2020 Global Game Jam". I participated at the New Orleans Global Game Jam site. I am pretty happy with the game I was able to create in two days with very little Unity knowledge. If you want to give the game a try click the link below!
  • Platform: WebGL
  • Release Date: 2 February 2020
  • Technologies: Unity5 Engine, C#, Aseprite
  • Play Here: http://www.marzgames.com/Builds/AstroRepair.html

AR Hanger

  • Technical Description: AR Hanger is an augmented reality application. It was developed for iOS using Apple's Swift programming language. It used multiple frameworks including Scenekit, ARKit, Spritekit, Gamekit. This was my second iOS application. I created this app to give myself some experience with app development that didn't consist of just video games.   • Release Date: 17 January 2017
  • Technologies: Swift, xCode, Scenekit, ARKit, Spritekit, Gamekit

multiple sites to get all of my space news. So I
built my own site that pulls data directly from many

There are no ads on my site. No distractions.
No Drama. Just Space News.

Astro Repair

Astro Repair is a game I created in under 48 hours
for Global Game Jam 2020. Realistically I developed
this game in 28~ hours. It is a 2d top down action
game where you play as one of the first Space Force
astronauts. You are stranded on a planet and have to
rescue your peers one at a time while fighting off
endless hordes of aliens.

AR Hanger

AR Hanger is the second iOS app I developed.
It is an app that uses Augmented reality technology that
allows you to take a picture of any item or download a
picture of an item from the internet. Then you can crop
that item in any shape needed. Once the item is cropped
you can virtually hang the item on the wall and walk
around the room to see what the item will look like.
